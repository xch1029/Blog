标签

如果你达到一个重要的阶段，并希望永远记住那个特别的提交快照，你可以使用 git tag 给它打上标签。
比如说，项目发布一个”1.0”版本。 我们可以用 git tag v1.0 命令给最新一次提交打上（HEAD）”v1.0”的标签。
